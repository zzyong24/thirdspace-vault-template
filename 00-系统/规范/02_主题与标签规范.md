# 主题与标签规范

## Topic

Topic 是唯一分类轴，用于回答“这个内容主要属于什么主题”。

核心 Topic 见 `.thirdspace/schema/taxonomy.yaml`。

## Tags

Tags 只表达：

- 内容类型
- 状态
- 来源
- 补充语义

## 命名规则

- 英文标签小写。
- 英文多词使用 kebab-case。
- 中文标签保持中文。
- 已有别名按 `.thirdspace/schema/taxonomy.yaml` 合并。

## Prompt 迁移

Prompt 不再作为全局内容类型维护。可复用提示词必须进入具体 Skill。
