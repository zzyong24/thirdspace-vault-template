# 01-收件箱 工作区规范

## 用途

接收外部输入和临时想法，等待整理进入知识、项目、资源或输出工作区。

## 允许

- Obsidian Web Clipper 抓取的网页内容
- 临时想法
- 待整理素材
- 未判断归属的输入

## 禁止

- 长期沉淀文件
- 已发布成品
- 完结项目归档

## 命名

所有新文件使用 `YYYYMMDD_主题.md`。网页剪藏放入 `网页剪藏/`，source 使用 `obsidian-clipper` 或 `web`。

## 清理规则

收件箱文件进入 `processed` 后，必须迁移到目标工作区或移入 `99-归档`。

## 子 Skill

使用 `workspace-inbox`。
